{{% pageinfo %}}
本文介绍文档教程的编写方式。
{{% /pageinfo %}}



每个目录下都需要创建一个 `_index.md` 文件。

文件内容开头：

```
---
title: "编辑文档"
linkTitle: "编辑文档"
weight: 10
description: >
  如何编辑修改本系列实践教程

---
```



子文件设计标签：

```
---
categories: ["Examples"]
tags: ["test", "sample", "docs"]
title: "Example Page"
linkTitle: "Example Page"
date: 2017-01-05
description: >
  A short lead description about this content page. It can be **bold** or _italic_ and can be split over multiple paragraphs.
---
```
